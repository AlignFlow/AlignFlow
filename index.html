Method Overview

Styled to match the pipeline figure: dashed frames + soft gradients.

Overview of the AlignFlow architecture. The observation is encoded to parameterize an observation-conditioned Gaussian mixture noise model, capturing discrete action modes and within-mode stochasticity.

During training, noise is sampled from all mixture components and actions are generated via mode-conditioned flow matching.

At inference, a single latent mode is selected conditioned on the observation, and the corresponding flow generates a trajectory-consistent action.

TRAIN
Sample noise from all mixture components and learn mode-conditioned flow matching.
INFERENCE
Select one latent mode conditioned on the observation; generate a coherent action via the corresponding flow.
